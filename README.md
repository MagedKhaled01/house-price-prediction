# house-price-prediction


This project explores predicting house prices using machine learning techniques on the Kaggle House Prices dataset.

Goal:

Build a model that can estimate house sale prices based on structural and quality-related features.

Approach:

The workflow followed an iterative modeling strategy:

Started with Linear Regression

Applied Ridge to stabilize feature influence

Used Lasso to detect weak predictors

Final model built using Gradient Boosting


 Result:

Final model
Gradient Boosting Regressor

Achieved:

0.14791 RMSLE on Kaggle public leaderboard

Tech Used:

Python

Pandas

Scikit-learn

Gradient Boosting

Output:

Submission file generated in Kaggle-required format:

Id | SalePrice
